# Data Folder

This folder contains Toyota car listing data collected via web scraping from
Craigslist New York, along with the notebook file used to generate the dataset.

## Contents

- **carScraping.ipynb**  
  A web scraping notebook used to collect Toyota car listings from
  Craigslist New York.  
  It includes data collection, cleaning, and basic preprocessing steps.

- **toyota_crap_data.csv**  
  The final dataset generated by running the `carScraping.ipynb` notebook.  
  This CSV file is cleaned and structured, making it suitable for
  price prediction and data analysis tasks.

## Dataset Features

The CSV file contains the following main columns:

- `Title` – Vehicle listing title  
- `year` – Vehicle model year  
- `Price` – Listing price (USD)  
- `condition` – Vehicle condition  
- `fuel` – Fuel type  
- `odometer` – Mileage information  
- `transmission` – Transmission type  
- `type` – Vehicle type  

## Notes

- The data was collected for educational and portfolio purposes.
- Source: Craigslist New York
- The dataset can be used for machine learning and car price prediction models.
